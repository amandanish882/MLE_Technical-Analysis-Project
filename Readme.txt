This package contains three files for the Capstone Project of Machine Learning Engineer Nanodegree: 

1: Proposal for the Project : Aman Danish Capstone_Proposal.pdf

2: Project Report : Aman Danish Capstone_Report.pdf

3: Data required to run the project : SNP 500 data.csv

4: The python notebook of the code : Analysis code_Final.ipynb


All the packages and libraries are mentioned in the Notebook and the References are mentioned in the Report.

To run the code, use SNP 500 data.csv as input data. All outputs are provided in the ipython notebook.
Code doesn't uses any external requirement.
